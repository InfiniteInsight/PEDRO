Task Completion Celebration

How should the confetti + dancing Pedro celebration work?

Confetti Animation

A

Canvas Confetti Library (Recommended)

Use a lightweight JS library like canvas-confetti for smooth, customizable particle effects. Colorful pieces shoot from bottom and fall naturally.

Pros

  • Smooth 60fps animation
  • Small file size (~3KB)
  • Theme-aware (colors match theme)
  • Performance optimized

Cons

  • Adds external dependency
  • Needs internet on first load
B

Pure CSS Confetti

Build confetti with CSS animations only — no external libraries. Simpler but less customizable.

Pros

  • Zero dependencies
  • Works offline immediately
  • Very lightweight

Cons

  • Less realistic motion
  • Harder to customize
  • Limited particle variety

Dancing Pedro Animation

C

Animated GIF Overlay

Show an actual GIF of the Pedro raccoon meme walking/dancing in front of the screen for 2-3 seconds when tasks are completed.

Pros

  • True to the actual meme
  • Maximum personality
  • Instantly recognizable

Cons

  • Need to find/create the GIF
  • Larger file size
  • Meme-dependent aesthetic
D

Bouncing Raccoon Emoji

Make the 🦝 emoji bounce, spin, and grow larger with CSS animations — keeps the single-file simplicity.

Pros

  • Zero file size increase
  • Works offline
  • Theme-consistent
  • Simple to implement

Cons

  • Not the actual meme
  • Less distinctive
E

CSS-Drawn Pedro Character

Create a simple stylized Pedro raccoon using pure CSS shapes that walks/dances across the screen.

Pros

  • Unique custom character
  • No images needed
  • Can be theme-aware

Cons

  • Time-intensive to design
  • Won't look like actual meme
